Privacy Policy for Calybridge

Last updated: 2026

Calybridge is a desktop application that allows users to synchronize calendar events between Yandex Calendar and Google Calendar.

1. Data Collection

Calybridge does not collect, store, or transmit personal data to external servers. All data processing happens locally on the user's device.

2. Google Data Access

Calybridge uses Google Calendar API to:

  • read calendar events
  • create events
  • update events
  • delete events

This access is granted by the user via Google OAuth authorization. Calybridge does not store or share this data with any third parties.

3. Data Storage

All data is stored locally on the user’s device, including:

  • authentication tokens
  • synchronization mappings

4. Third-Party Services

Calybridge uses:

  • Google APIs (Google Calendar)
  • Yandex CalDAV

5. Security

Calybridge does not transmit user data to external servers.
