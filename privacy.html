Privacy Policy

Calybridge is a desktop application that synchronizes calendar data between Yandex Calendar and Google Calendar.

Data Collection

Calybridge does not collect or store personal data on external servers.

Google Data Access

The application accesses Google Calendar data only after user authorization via OAuth.

Data Storage

All data is stored locally on the user's device.

Third-party services

Calybridge uses Google Calendar API and Yandex CalDAV.
